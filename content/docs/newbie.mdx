# 小白AI-Web3共学营

## 为什么非技术人员也能参与AI+Web3

AI+Web3的融合已经大大降低了技术门槛，即使没有编程背景，也可以通过各种工具和平台参与到这个领域中来。本指南专为非技术背景的学习者设计，帮助你:

- 理解AI和Web3的基本概念和价值
- 学习使用无代码工具构建简单应用
- 掌握AI提示词技巧，让AI成为你的"编程助手"
- 找到适合自己的参与路径和贡献方式

## 入门学习路径

1. **基础概念学习** (1-2周)
   - 了解区块链和Web3的基本原理
   - 熟悉AI大模型的能力和局限性
   - 学习钱包创建和基本操作

2. **工具探索阶段** (2-4周)
   - 尝试使用AI工具（如ChatGPT、Midjourney）
   - 探索无代码Web3平台（如thirdweb、Buildspace）
   - 参与简单的Web3项目体验（如铸造NFT）

3. **实践项目阶段** (1-2个月)
   - 设计一个小型AI+Web3创意项目
   - 参与AI³ Growth Journey的线上workshop
   - 和有经验的开发者合作，将创意变为现实
