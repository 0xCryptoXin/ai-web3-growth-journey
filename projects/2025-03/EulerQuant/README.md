# AI创意漫画NFT

## 项目介绍
### 背景与目标
通过结合 AI 生成艺术与区块链技术，打造个性化 NFT 生成平台。用户输入文字描述后，由 AI 生成独特数字艺术作品，并一键铸造为 NFT 上链存储。项目采用模块化架构，重点实现：
- AI 图像生成接口
- 区块链智能合约交互
- 极简主义风格前端界面

### 核心功能
1. 文本到图像转换（Text-to-Image）
2. NFT 元数据标准化存储
3. 链上铸造与所有权验证
4. 实时交易状态追踪

### 技术亮点
- 使用 **ThirdWeb TypeScript SDK** 实现智能合约快速接入
- 基于 **Monad Testnet** 的低成本测试环境
- 采用 **Tachyons CSS** 的原子化样式设计
- 响应式布局适配多端设备
