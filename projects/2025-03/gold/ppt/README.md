# 去中心化创意内容平台演示

这是一个基于HTML的演示网站，用于展示去中心化创意内容平台的产品理念和功能特点。

## 项目结构

```
/
├── index.html          # 主入口文件
├── slides/            # 幻灯片文件
│   ├── intro.html     # 产品介绍
│   ├── product.html   # 核心功能
│   ├── features.html  # 用户旅程
│   ├── tech.html      # 技术架构
│   ├── advantage.html # 差异化优势
│   └── contact.html   # 运营策略
├── css/               # 样式文件
│   └── style.css      # 主样式文件
└── js/                # JavaScript文件
    └── main.js        # 主脚本文件
```

## 功能特点

- 响应式设计，适配桌面和平板设备
- 现代化UI设计，使用Tailwind CSS
- 流畅的动画效果和过渡
- 支持键盘导航（左右方向键）
- 全屏显示功能
- 左侧导航菜单（可隐藏）
- 幻灯片导航指示器

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 使用以下方式导航：
   - 点击左侧菜单项
   - 使用键盘左右方向键
   - 点击右下角导航点
3. 使用右上角按钮进入/退出全屏模式
4. 使用左上角按钮显示/隐藏导航菜单

## 技术栈
